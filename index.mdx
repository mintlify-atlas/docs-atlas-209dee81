<Note>
  This project is **not** associated with the official i-Ma'luum or IIUM. It is an unofficial, community-built API wrapper.
</Note>

GoMa'luum is an open-source proxy API that wraps IIUM's i-Ma'luum student portal, returning clean JSON for student academic data.
