---
title: 'Generate API key'
description: 'Generate a timestamped API key for use as an additional security layer on login and authenticated requests.'
api: 'POST https://api.quddus.my/api/key/generate'
---

Generates a new API key in the `gml_<timestamp>_<hex>` format. Pass this key in the `x-gomaluum-key` header when calling [POST /api/auth/login](/api-reference/auth/login). The same key must then be included on every subsequent authenticated request made with the resulting token.

<Note>
  The `x-gomaluum-key` header is optional. If you do not supply one, the server falls back to a default internal key. Generating your own key provides an additional, caller-controlled security layer.
</Note>

## API key format

Generated keys follow the pattern:

```
gml_<unix-timestamp>_<64-char-hex>
```

Example: `gml_1710000000_a3f9c2b1d4e8f0a1b2c3d4e5f6a7b8c9d0e1f2a3b4c5d6e7f8a9b0c1d2e3f4a5`

The prefix `gml_` identifies the key as a GoMa'luum API key. The Unix timestamp (in seconds) records when the key was created. The 64-character hex string encodes 32 bytes (256 bits) of cryptographically random data generated with `crypto/rand`.

## Request

No request body or authentication is required.

## Response

<ResponseField name="api_key" type="string">
  The generated API key in `gml_<timestamp>_<hex>` format. Store this value securely — it is not persisted server-side.
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable instructions on how to use the key.
</ResponseField>

<ResponseField name="created_at" type="string">
  Timestamp indicating when the key was generated.
</ResponseField>

## Error responses

| Status | `message`                    | Cause                                         |
| ------ | ---------------------------- | --------------------------------------------- |
| `500`  | `Failed to generate API key` | Internal error during key generation.         |

```bash cURL
curl --request POST \
  --url https://api.quddus.my/api/key/generate
```

```javascript JavaScript
const response = await fetch('https://api.quddus.my/api/key/generate', {
  method: 'POST',
});

const data = await response.json();
const apiKey = data.api_key;
```

```python Python
import requests

response = requests.post('https://api.quddus.my/api/key/generate')
api_key = response.json()['api_key']
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    resp, err := http.Post(
        "https://api.quddus.my/api/key/generate",
        "application/json",
        nil,
    )
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    var result map[string]any
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Println(result["api_key"])
}
```
