---
title: "Get exam slip"
description: "Stream the authenticated student's exam slip as a PDF from i-Ma'luum."
---

<Note>
  This endpoint requires a valid IIUM session. Obtain a token by authenticating with `POST /api/login` first.
</Note>

## Request

**`GET https://api.quddus.my/api/download/exam-slip`**

Streams the student's exam slip directly from i-Ma'luum as a binary PDF. The response body is the raw PDF file.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from the login endpoint. Format: `Bearer <token>`
</ParamField>

<ParamField header="x-gomaluum-key" type="string">
  Optional API key for an additional security layer. Required only if the server is configured to enforce API key authentication.
</ParamField>

## Response

On success, the server responds with HTTP `200` and streams the PDF file. The response has no JSON wrapper — the body is the raw binary PDF content.

| Header | Value |
|---|---|
| `Content-Type` | `application/pdf` |

## Error responses

| Status | Message | Cause |
|---|---|---|
| `401` | `Invalid token` | The `Authorization` header is missing or the token is expired |
| `401` | `Invalid API key` | The `x-gomaluum-key` value does not match the configured key |
| `500` | `Failed to parse URL` | Internal request to i-Ma'luum could not be constructed |
| `500` | `Failed to download the file` | The PDF stream from i-Ma'luum could not be copied to the response |

## Examples

<CodeGroup>

```bash cURL
curl --request GET \
  --url https://api.quddus.my/api/download/exam-slip \
  --header 'Authorization: Bearer <token>' \
  --output exam-slip.pdf
```

```javascript JavaScript
const response = await fetch('https://api.quddus.my/api/download/exam-slip', {
  headers: {
    Authorization: 'Bearer <token>',
  },
});

if (!response.ok) {
  throw new Error(`Request failed: ${response.status}`);
}

const blob = await response.blob();
const url = URL.createObjectURL(blob);

// Trigger download in the browser
const a = document.createElement('a');
a.href = url;
a.download = 'exam-slip.pdf';
a.click();
URL.revokeObjectURL(url);
```

</CodeGroup>
