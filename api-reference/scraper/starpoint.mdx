---
title: "Get starpoint"
description: "Scrapes co-curricular starpoint records from i-Ma'luum."
api: "GET https://api.quddus.my/api/starpoint"
---

Fetches the student's co-curricular activity starpoint records, including individual programme entries and the overall cumulative average.

## Request

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from the login endpoint. Format: `Bearer <token>`
</ParamField>

<ParamField header="x-gomaluum-key" type="string">
  Optional API key for an additional security layer. Required when the server is configured to enforce API key authentication.
</ParamField>

## Response

<ResponseField name="message" type="string">
  Human-readable status message. Value is `Successfully fetched starpoints programs` on success.
</ResponseField>

<ResponseField name="data" type="object">
  The starpoint record for the student.
  <Expandable title="Starpoint properties" defaultOpen>
    <ResponseField name="id" type="string">
      Unique identifier for this starpoint record, prefixed with `gomaluum:starpoint:`.
    </ResponseField>
    <ResponseField name="total_points" type="number">
      Sum of all starpoints earned across all programmes and semesters.
    </ResponseField>
    <ResponseField name="cummulative_average" type="number">
      Cumulative starpoint average as shown in i-Ma'luum. Note: the field name reflects the spelling used in the source portal.
    </ResponseField>
    <ResponseField name="programs" type="StarpointProgram[]">
      Array of individual co-curricular programme entries.
      <Expandable title="StarpointProgram properties">
        <ResponseField name="id" type="string">
          Unique identifier for this programme entry, prefixed with `gomaluum:program:`.
        </ResponseField>
        <ResponseField name="session" type="string">
          Semester during which this programme took place (e.g. `2023/2024 - 1`).
        </ResponseField>
        <ResponseField name="event_name" type="string">
          Name of the co-curricular event or programme.
        </ResponseField>
        <ResponseField name="type" type="string | null">
          Category or type of the programme. May be `null` if not specified in i-Ma'luum.
        </ResponseField>
        <ResponseField name="level" type="string">
          Participation level (e.g. `College`, `National`, `International`).
        </ResponseField>
        <ResponseField name="points" type="number">
          Starpoints awarded for this programme entry.
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>
