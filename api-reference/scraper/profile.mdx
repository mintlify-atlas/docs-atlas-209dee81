---
title: "Get profile"
description: "Scrapes and returns the student profile from i-Ma'luum."
api: "GET https://api.quddus.my/api/profile"
---

## Request

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from the login endpoint. Format: `Bearer <token>`
</ParamField>

<ParamField header="x-gomaluum-key" type="string">
  Optional API key for an additional security layer. Required when the server is configured to enforce API key authentication.
</ParamField>

## Response

<ResponseField name="message" type="string">
  Human-readable status message. Value is `Successfully fetched profile` on success.
</ResponseField>

<ResponseField name="data" type="object">
  The student profile object.
  <Expandable title="properties" defaultOpen>
    <ResponseField name="image_url" type="string">
      URL of the student's ID card photo hosted on the IIUM smartcard server.
    </ResponseField>
    <ResponseField name="name" type="string">
      Full name of the student as registered with IIUM.
    </ResponseField>
    <ResponseField name="matric_no" type="string">
      Student matriculation number (e.g. `2210893`).
    </ResponseField>
    <ResponseField name="level" type="string">
      Current level of study (e.g. `Degree`).
    </ResponseField>
    <ResponseField name="kuliyyah" type="string">
      Name of the kulliyyah (faculty/college) the student is enrolled in.
    </ResponseField>
    <ResponseField name="ic" type="string">
      Identity card number.
    </ResponseField>
    <ResponseField name="gender" type="string">
      Student gender (e.g. `Male`, `Female`).
    </ResponseField>
    <ResponseField name="birthday" type="string">
      Date of birth as a formatted string (e.g. `01-01-2000`).
    </ResponseField>
    <ResponseField name="religion" type="string">
      Religion as recorded in the student record (e.g. `Islam`).
    </ResponseField>
    <ResponseField name="marital_status" type="string">
      Marital status (e.g. `Single`, `Married`).
    </ResponseField>
    <ResponseField name="address" type="string">
      Permanent home address, formatted as a comma-separated single-line string.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://api.quddus.my/api/profile \
  --header 'Authorization: Bearer <token>'
```

```javascript JavaScript
const response = await fetch('https://api.quddus.my/api/profile', {
  method: 'GET',
  headers: {
    Authorization: 'Bearer <token>',
  },
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    'https://api.quddus.my/api/profile',
    headers={'Authorization': 'Bearer <token>'},
)

print(response.json())
```
